Map of Howe Sound, BC

This project was part of a BCIT cartography course and involved designing a reference map by implementing one's own specifications for cartographic elements like symbols, colours, labels, layout, etc.

A couple of things I would improve upon if I were to do it again would include perhaps playing around with the elevation contour colour a bit more and improving the road layers in the key map. Otherwise I found the results to be very satisfying and I quite enjoyed the artistic process of designing this map.
